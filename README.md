# Lung X-Ray Classification using ResNet34

## Overview

This project implements a deep learning model for classifying lung X-ray images using transfer learning with ResNet34.

The model was developed with PyTorch and evaluated using 10-Fold Cross Validation to ensure reliable performance.

## Features

- ResNet34 Transfer Learning
- PyTorch
- Data Augmentation
- Fixed Random Seeds
- 10-Fold Cross Validation
- Google Colab (NVIDIA T4 GPU)

## Results

- Accuracy: 96.8% - 97.1%
- Runtime: Approximately 2–3 minutes
- Epochs: 12
- Batch Size: 32

## Technologies

- Python
- PyTorch
- Torchvision
- NumPy
- Scikit-learn

## Dataset

The dataset is not included in this repository due to size limitations.
